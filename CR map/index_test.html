Erasmus Mundus Course Web Map

The Erasmus Mundus Association is a professional network for Erasmus Mundus students and graduates. This webmap serves as a search tool for graduate programs by location and links prospective students to program webpages.

There are two modes: Program and City.

  • The Program mode allows one to filter the programs in mapview by program name and displays markers for all locations of an individual program.
  • The City displays programs and links to individual webpages by city.
The default filter for Degree and Status buttons were set up to All, which displays both Master and Doctorate programs.
  • The Degree button enables selection for Masters, Doctorates or ALL programs, and
  • the Status button enables selection for Discontinued or Ongoing programs, whereas:
    • Discontinued is an indicator for programs which do not receive fundings by European Commision any longer, while
    • Ongoing are active programs for the ongoing year. Totally new programs funded by European Commision first time ever were marked with NEW
For more details, please see the list of all Erasmus Mundus Masters Courses.

We hope that this program exploration tool will serve as a resource for students, faculty, and EMA staff.

Read more details at the EMA website.
